# Pizza Ordering App

This project is a simple pizza ordering application that simulates a real-world ordering system.

## 🚀 Features

- Browse different pizza types  
- Add pizzas to cart  
- View current order (cart)  
- Display quantity of each pizza  
- Remove last added item  
- Remove all items of a selected type  
- Calculate total price dynamically  

## 🧠 What I Learned

Through this project, I worked with:
- Managing application state using arrays  
- Handling user interactions with event-driven programming  
- Structuring and updating dynamic data (cart system)  
- Using loops and logic to calculate totals and quantities  
- Improving user experience with better feedback and functionality  

## 🎯 Purpose

The goal of this project was to simulate a simple food ordering system and improve on earlier projects by adding more advanced functionality such as cart management and item removal.

## 🔧 Example Output

The cart displays items in the following format:

Skinke x2 - 90 kr  
Pepperoni x1 - 55 kr  

Total price is calculated and displayed separately.
